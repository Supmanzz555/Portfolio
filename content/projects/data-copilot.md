## Overview

A Retrieval-Augmented Generation (RAG) copilot system designed to help users answer SQL-related questions by combining the power of LLMs with database knowledge retrieval.

## Key Features

- RAG-based question answering for SQL queries
- Context-aware SQL query generation
- Knowledge retrieval from documentation and schemas
- Interactive query refinement
- Support for multiple database dialects
