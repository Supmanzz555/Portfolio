## Overview

A real-time data pipeline for processing e-commerce clickstream events. Built with Kafka for ingestion and Spark for stream processing.

## Key Features

- Real-time click event ingestion via Apache Kafka
- Stream processing with Apache Spark
- E-commerce user behavior analytics
- Scalable event-driven architecture
